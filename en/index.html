Built for Non-technical User

Designed for non-technical users at the first place. It keeps the technical barriers very low by providing a drag and drop interface to build both ETL and analysis.

Powered by SQL

Make it possible to query, transform and analyze with standard SQL for not only tranditional relational database but also other data sources, for example, file.

Offline Mode

Built-in analysis execution engine makes it possible to run your analysis job on your own computer without internet connection.

Collaborate with Engineers

Comes with technical tools for engineers to test, and run Ananas data projects in cloud or on premise. Share preconfigured project with non-technical users.

Scalable

Ananas Desktop has been used on production for both small ad-hoc queries and terabyte big data analysis

Connect data from anywhere in any format

  • Connect to relational database, NoSQL, File, API, and more ...
  • Support both structured and unstructured data
  • Join and concatenate data from different sources

Avoid the hassle of complex SQL or Python

  • Minimal knowledge required for Simple SQL
  • User friendly desktop interface

Visualize with few clicks
